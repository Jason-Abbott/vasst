Creating a new web using a web template

1. On the File menu, go to New, and then Web. (For FrontPage 2002, on the File menu, go to New, and then Web. Or, on the taskbar to the right, choose "Web Site Templates" under "New From Template.")

2. Select the template you want to use to create the new web from the Web Sites list.

3. In the Specify the location of the new web box, specify the location where you want to publish your new web.

Adding a new page to using a page template

1. On the File menu, go to New, and then Page. (For FrontPage 2002, on the File menu, go to New, and then Page. Or, on the taskbar to the right, choose " Page Templates" under "New From Template.")

2. Choose the page template from the list of page templates.

3. Click OK.

4. Save the page into your web and any embedded images into the "images" folder.

5. Go to Navigation View and drag your page into your navigation structure.

Deleting a page

In Folder List, right-click the page you want to delete and click "Delete."

Renaming a page

In Navigation view, right-click the page for which you want to change the label, click Rename on the shortcut menu, and then edit the name.

Changing the flash text in .swi file

1. Open the original .swi source file in Swish program.

2. In Outline panel to the left, Double click the text you like to modify.

3.The Text panel will appear on the right. In Text panel, change the text, font size and color as you want.

4. Save the file.

5. On the File menu, go to Export and then click SWF or Ctrl+E. The .swf file will be automatically created)

Importing the modified flash movie into your web

1. In Normal view in FrontPage, select the current flash movie and delete.

2. On the Insert menu, click Web Component.

3. In the left pane, click Advanced Controls.

4. In the right pane, double-click Plug-In.

5. In the Data source box, type the URL of the file (your modified .swf file) for the plug-in to load, or click Browse to locate the file.

Size : To specify the width and height (in pixels) of the plug-in, specify the desired value in the Size box.
